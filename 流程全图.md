
## ComfyMuse 完整流程图（Full Flowchart）

**ASK 交互规则**（全流程通用）：
- AI 优先从 SKILL.md 题库（分类/维度/NSFW）选取选项 → 省 token
- 题库没有时 → AI 自主生成，并保存回题库（越用越聪明）
- 每轮选项下方标注：⚠️ 或在上方「输入你的答案」框自行补充
- 生成的新内容 → SFW 存 Addons-SFW.md / NSFW 存 Addons-NSFW.md

## 术语对照表

| 缩略语 | 全称 |
|--------|------|
| SFW | Safe For Work — 安全内容 |
| NSFW | Not Safe For Work — 成人内容 |
| Workbench | 工作台.md — 当前方案草稿 |
| Preview | 预览图.png — 当前生成的测试图 |
| Versions | 版本记录/ — 历史版本归档 |
| Core | 核心配置.md — Obsidian 偏好记录 |
| Cases | 有效案例/ — Obsidian 成功方案 |
